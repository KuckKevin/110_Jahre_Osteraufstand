Pushen auf GitHub 
GitHub nutzt Personal Access tokens um Commits zu pushen (auch wenn dort "Password" steht).
Diesen erstellt man so:
GitHub → Profilbild → Settings → Developer settings (ganz unten)
→ Personal access tokens → Fine-grained tokens → Generate new token

Wichtig: Token kopieren, da man diesen nur einmal sieht.

Beim Push dann den Token anstatt des Passworts verwenden.
